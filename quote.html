Details that make the first response more useful.

A quote request does not need to be perfect, but it should give Highline enough context to avoid guessing.

Location

Include the address, city, county, or nearest service area. Jacksonville, the beaches, Clay County, Nassau County, Baker County, and St. Johns County can involve very different travel and coverage assumptions.

Service type

Pick the closest service, even if you are unsure. Guards, mobile patrols, fire watch, event security, alarm response, consulting, and apartment security each start from a different operating plan.

Schedule

Share the date, start time, duration, recurring schedule, or temporary coverage window. Timing helps Highline understand urgency and staffing fit.

Concerns

Explain what prompted the request: access issues, recent incidents, fire-system concerns, resident complaints, event crowd flow, after-hours activity, or the need for stronger reporting.

Decision maker

Include who should approve the plan, who receives reports, and whether the quote should account for owner, property manager, event organizer, or facility-team requirements.
